# LongShopV3

主要是用原Vue3的版本进行商店的编辑用，外加一些功能，这个是所有的引用的开始

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
